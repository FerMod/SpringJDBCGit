DEFINICIÓN DEL PROYECTO

Hay que hacer el programa de mantenimiento de una tabla de la afamada Base de Datos Almacén. Cada grupo lo hará de una tabla. 
Las tablas asignadas a cada grupo son:

Grupo 1: Tabla Artículos.

Grupo 2: Tabla Albaranes.

Grupo 3: Tabla Clientes.

Grupo 4: Tabla Proveedores.

Grupo 5: Tabla Líneas.


Para cada tabla se pide lo siguiente:

	1.	Añadir nuevos registros a la tabla.

	2.	Modificar el contenido de registros.
		a.	Se introducirá la clave primaria.
		b.	Se sacarán los datos por pantalla.
		c.	Se introducirán los datos para modificar.
		d.	Se procederá a la modificación.

	3.	Borrar registros de la tabla.
		a.	Se introducirá la clave primaria.
		b.	Se sacarán los datos por pantalla.
		c.	Se pedirá conformidad para borrar.
		d.	Se procederá al borrado.


	4.	Consultar registros de la tabla.
		a.	Se introducirá la clave primaria.
		b.	Se sacarán los datos por pantalla.
